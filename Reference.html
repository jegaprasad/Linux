Chmod ----to change the file permission


Chmod 666 filename
Chmod 755 file name
Chmod 750 filename

Ownership change


Chown sasi:sasi filename
Chown sasi:root filename | Owner Group

Now change the group name


Chown sasi:root filename
Now change the owner name
Chown root:sasi filename

Shortcut


Chown :root filename ---it will consider after : that is group so it will change group permission only
Chown root filename ----not it will consider root is before : so only owner name will change
Inside the folder to change the all permission use -R
Chmod -R 755 foldername
Chown -R root:root foldername

Remove the permission:


Chmod u-x filename
Chmod u+x filename
Chmod g+x filename

To change all


chmod a-X filename to remove the execute permission to all user,group and others
