Linux Directory Structure – 20 MCQs

1. What is the purpose of the /etc directory?
A. Contains temporary files
B. Contains user data
C. Contains configuration files and scripts
D. Contains boot files
✅ Answer: C
2. Which directory stores system log files?
A. /usr/log
B. /var/log
C. /tmp/log
D. /etc/logs
✅ Answer: B
3. Where is the Linux kernel typically located?
A. /lib
B. /boot
C. /kernel
D. /etc
✅ Answer: B
4. What does the /home directory store?
A. System binaries
B. User home directories
C. Configuration files
D. Shared libraries
✅ Answer: B
5. What is the /root directory?
A. Root of the filesystem
B. User login directory
C. Superuser’s home directory
D. Kernel directory
✅ Answer: C
